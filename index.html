Verið velkomin

Á heimasíðuna mína fyrir námskeiðið Tölvustudd framleiðsla á vorönn 2023

Hlutverk þessarar vefsíðu er að halda utan um verkefni og vinnuframlag í námskeiðinu Tölvustudd framleiðsla við Háskóla Íslands. Í námskeiðinu eru kennd undirstöðuatriði tölvustuddrar framleiðslu og stafrænnar framleiðslutækni. Í námskeiðinu verður meðal annars farið í þrívíða hönnun fyrir 3D-prentun og 3D-skönnun, parametríska hönnun fyrir geislaskurð og fræsingu og hönnun á útsteypimótum. Notast verður við forritunarmálin HTML og CSS til að vinna í þessari vefsíðu og Git verður notað til að hýsa þessa síðu og skrásetja vinnuframlag mitt.
